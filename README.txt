Задача парсера - выгрузка изображений с сайта "https://joyreactor.cc/tag/%D0%BC%D0%B8%D0%BB%D0%BE%D1%82%D0%B0" 
и их систематизация путём подписи в соответствии с набором тегов и рейтингом.

Для запуска парсера необходимо дважды кликнуть по файлу main.py
После запуска парсер в файле outputs создаёт файл, по названию идентичный разделу сайта.
Затем в созданный файл загружаются изображения и подписываются в соответствии со своим набором тегов и рейтингом.
